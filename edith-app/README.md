# E.D.I.T.H - AI Assistant

**Even Dead, I'm The Hero**

A Spider-Man themed AI voice assistant with React, TypeScript, and Electron.

## Quick Start

```bash
# Install dependencies
npm install

# Run web version
npm run dev

# Run desktop app
npm run electron:dev

# Build for production
npm run build
npm run electron:build
```
