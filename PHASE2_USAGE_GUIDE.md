## ğŸ› Troubleshooting

### Personality Not Changing
- Check for red border on selected card
- Look for pulsing dot indicator
- Check activity feed for confirmation
- Try clicking again

### Quick Commands Not Working
- Must start with `/` character
- Check spelling: `/time` not `/times`
- Use `/help` to see all commands
- Check activity feed for errors

### Stats Not Updating
- Refresh page to reload
- Check browser cookies enabled
- Clear cookies and restart
- Stats reset on cookie clear

- Scroll to top of page
- Check screen width (responsive)
- Refresh page
- Check browser console for errors
